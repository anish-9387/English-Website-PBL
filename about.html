About This Website


This website has been developed as part of the English Project-Based Learning (PBL) activity conducted for Class X-1 students. The aim of the activity is to enhance students' communication skills, presentation abilities, and collaborative spirit through engaging, hands-on projects.

Each student was assigned to form a team and worked together on SDGs (Sustainable Development Goals) relevant to their chosen book. As part of the project, teams were asked to research their topics, prepare a report (PDF), design a presentation (PPT), and record an explanatory video.


To bring all of this together in one place, this website was created using HTML, CSS, and JavaScript. The home page features a responsive grid layout showcasing all teams. When a user clicks on a group, a modal window opens with that team's name, list of members, and three action buttons: Report, PPT, and Video.

Each of these buttons loads the corresponding resource from the group: the PDF report, the PowerPoint presentation, and the YouTube video link. This ensures that all project material is centralized in one easy-to-navigate interface. The design is responsive for both desktop and mobile views, with a clean user interface and modal system for content viewing.

The files are hosted on Google Drive and embedded using secure links to ensure smooth viewing without the need for downloading. This makes the platform lightweight, efficient, and easy for teachers and students to use.


In summary, this site not only showcases the hard work and creativity of all teams involved but also demonstrates how digital platforms can be effectively used for academic collaboration and skill-building.
